Balloon Pop Maths

Belajar Matematika Jadi Lebih Seru!

Aplikasi pembelajaran matematika interaktif untuk siswa SD dengan konsep permainan balon yang menyenangkan. Tersedia dalam 2 versi: Aplikasi Desktop dan Mobile App.

Pilih Versi Aplikasi

Aplikasi Desktop

Download dan install di komputer Windows. Bisa digunakan tanpa koneksi internet.

  • Instalasi mudah dengan setup wizard
  • Performa lebih cepat
  • Desktop & Start Menu shortcut
Windows 10/11 ~120 MB
Download untuk Windows

Versi 1.0.0

Aplikasi Mobile

Download dan instal di handphone anda pastikan anda memiliki koneksi internet.

  • Bisa dari HP/Tablet
  • Akses dari mana saja
Android 15 ~16 mb
Download Untuk Android

Cara Instalasi

Aplikasi Desktop (Windows)

  1. Klik tombol "Download untuk Windows"
  2. Tunggu download selesai (~120 MB)
  3. Double-click file "Balloon Pop Maths Setup.exe"
  4. Ikuti wizard instalasi
  5. Klik icon di desktop untuk memulai

Aplikasi Mobile

  1. Klik tombol "Download Untuk Android"
  2. Tunggu download selesai (~16 MB)
  3. Setelah itu install di hp
  4. Pilih mode permainan yang diinginkan
  5. Klik "Play" dan mulai bermain
